Our Process

How the Assessment Works

01

Scope Definition & Document Review

We identify your CUI boundary, review existing security documentation (policies, procedures, network diagrams), and establish the system scope for assessment. This determines which of the 110 controls apply and how they should be evaluated.

02

Control-by-Control Assessment

We evaluate every NIST 800-171 control through a combination of documentation review, technical configuration verification, and personnel interviews. Each control receives a status: Implemented, Partially Implemented, Not Implemented, or Not Applicable — with specific evidence notes.

03

SPRS Score Calculation & Report Delivery

We calculate your Supplier Performance Risk System score using the DoD Assessment Methodology. SPRS scores range from -203 (no controls implemented) to 110 (full compliance). The current median SPRS score across the DIB is estimated at 40–60 per DCMA assessment data. We deliver a comprehensive report with findings organized by control family, your exact score with a control-by-control breakdown, and a prioritized remediation roadmap weighted by both risk severity and implementation cost with timeline recommendations.
